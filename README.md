# Pencil

Setting up
==========

```bash
git clone https://github.com/wtasg/pencil.git
cd pencil
npm install
npm run start
```

Testing
=======

Unit tests:

```bash
npm test
```

Electron E2E smoke tests (Playwright):

```bash
npm run test:e2e
```

Run headed locally:

```bash
npm run test:e2e:headed
```
